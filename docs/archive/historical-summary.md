# 历史版本汇总 (1.0.x - 1.2.x)

## 1.2.13 - 2026-04-25

### 变更类型

Bug 修复

### 变更内容

- 修复 Redis Lock 无法上锁和上锁失败的问题
- 调整兼容性，修复部分名称使用默认名称导致 bean 名称重复的问题

---

## 1.2.11 - 2026-04-15

### 变更类型

Bug 修复

### 变更内容

- 修复日志记录器的性能问题
- 修复 MinIO 存储模块的连接池问题

---

## 1.2.0 - 2026-01-01

### 变更类型

功能新增

### 变更内容

- 添加 MinIO 对象存储支持
- 添加 Redisson 分布式锁支持
- 添加日志记录模块
- 升级至 Spring Boot 2.7.x

---

## 1.1.0 - 2025-06-01

### 变更类型

功能新增

### 变更内容

- 添加 Redis 缓存支持
- 添加 MyBatis Plus 支持

---

## 1.0.0 - 2025-01-01

### 变更类型

初始版本

### 变更内容

- 项目初始化
- 基础模块结构搭建
- RESTful Web 支持
- MyBatis 支持
