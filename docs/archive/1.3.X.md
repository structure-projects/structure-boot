## 组件使用指南

### Web 开发组件

#### structure-restful-web-starter

提供 RESTful API 开发支持，包括统一异常处理、参数校验、Swagger 文档等。

**依赖引入:**

```xml
<dependency>
    <groupId>cn.structured</groupId>
    <artifactId>structure-restful-web-starter</artifactId>
</dependency>
```

**功能特性:**

| 功能 | 说明 |
| :--- | :--- |
| 统一异常处理 | `@EnableSimpleGlobalException`: 简易版异常处理 |
| 统一返回结果 | `ResResultVO<T>`: 简易版返回结果<br>`ResultVO<T>`: 二级码返回结果 |
| Swagger 文档 | `@EnableSwagger`: 开启 Swagger |

---

### 数据访问组件

#### structure-mybatis-starter

提供 MyBatis 的增强功能，包括自动 ID 生成、创建/更新时间注入等。

**配置参数:**

| 参数 | 说明 | 默认值 |
| :--- | :--- | :--- |
| `generate-id-type` | ID生成类型: none, uuid, snowflake | snowflake |
| `data-center` | 数据中心码 (0-31) | 0 |
| `machine` | 机器码 (0-31) | 0 |

**配置示例:**

```yaml
structure:
  mybatis:
    plugin:
      generate-id-type: snowflake
      data-center: 0
      machine: 0
```

---

### 缓存组件

#### structure-redis-starter

提供基于 Redis 的分布式锁实现。

**注解方式:**

```java
@RedisLock("#orderId")
public void processOrder(String orderId) {
    // 业务逻辑
}
```

#### structure-redisson-starter

基于 Redisson 的高级缓存功能。

**注解方式:**

```java
@WCache(key = "#user.id", isObjCache = true)
public User cacheUser(@RequestBody User user) {
    return user;
}
```

---

### 文件存储组件

#### structure-minio-starter

提供 MinIO 对象存储的集成支持。

**配置:**

```yaml
structure:
  minio:
    url: http://localhost:9000
    access-key: your-access-key
    secret-key: your-secret-key
    endpoint-enable: true
```

## 版本升级建议

### 从 1.2.x 升级到 1.3.7

**主要变更:**

- Spring Boot 2.x → 3.2.x
- JDK 8/11 → 17/21
- Jakarta EE 8 → Jakarta EE 9+

**注意:**

1. 数据库驱动可能需要更换（如 `mysql-connector-java` → `mysql-connector-j`）
2. `spring.redis` 配置迁移至 `spring.data.redis`
3. 部分 SpEL 表达式语法可能有变化
4. 需要使用 `jakarta.*` 包名替代 `javax.*`
