## 版本概述

v1.4.1 是依赖版本的小幅升级版本，主要更新 Spring Boot 和 Knife4j 的版本。

### 主要更新

- **Spring Boot 版本小版本升级**: 4.0.0 → 4.0.6
- **Knife4j 版本升级**: 4.4.0 → 4.5.0
- 修复了部分 starter 模块在 Spring Boot 4.x 环境下的兼容性问题
